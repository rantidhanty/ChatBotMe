# 02 - Memahami Struktur Awal Next.js

## Tujuan

Dokumen ini menjelaskan struktur awal proyek Next.js yang dibuat di folder `web/`.

## Struktur Level Atas

Di root repository saat ini ada:

- `CLAUDE.md`
- `README.md`
- `docs/`
- `web/`

## Struktur di Dalam `web/`

Folder dan file penting:

- `package.json`
- `README.md`
- `AGENTS.md`
- `src/`
- `public/`

## Fokus Belajar Saat Ini

- fungsi `package.json`
- fungsi `layout.tsx`
- fungsi `page.tsx`
- perbedaan `globals.css` dan `page.module.css`
