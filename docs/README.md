# Documentation Index

Dokumen di folder ini dibuat untuk membantu memahami proyek `ChatBotMe` langkah demi langkah.

## Aturan Belajar Proyek

- setiap penjelasan teknis penting akan dibuat juga dalam file `.md`
- dokumentasi dipisah per jalur agar materi tidak tercampur
- urutan dokumen dibuat bertahap agar mudah dipelajari
- dokumen akan fokus pada alasan, fungsi, dan cara kerja, bukan hanya hasil akhir
